Laravel 5 Wrapper for the eluceo/ical Package
=====================================

This is a wrapper for the eluceo/ical package. 

How to Install
---------------

### Laravel 5.+

1.  Install the `nikjaysix/laravel-ical` package

    ```shell
    $ composer require nikjaysix/laravel-ical
    ```
